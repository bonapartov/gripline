# Аналитический модуль для картинговой статистики

## Структура проекта

- `core/` — базовые утилиты для загрузки и обработки данных
- `bradley_terry/` — реализация модели Брэдли-Терри с Lasso-регуляризацией
- `pagerank/` — модифицированный PageRank (в разработке)
- `ensemble/` — ансамбль моделей (в разработке)
- `context/` — учет погоды и шин (в разработке)
- `scripts/` — management-команды для Django
- `tests/` — тесты

## Использование

### Обновление рейтингов шасси

python manage.py update_ratings --entity chassis --alpha 0.1
